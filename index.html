Slap Cost

CMPS 285- Software Engineering

Slap cost is an Antroid application designed for a construction company created using Xamarin forms. In the job estimate page, you may enter dimensions and job-types, calculate cost based on company information sheets and mark created jobs as accepted, pending or paid. And in the customer page the list of customers with their contact details and addresses are contained.

Technologies used: C#, Xamarin forms, SQLite, Visual Studio


Nexus Shelf

CMPS 401- Survey of Programming languages

Nexus Shelf is an ecommerce website used for trading books built with Ruby on Rails. Interested users can create an account and sign in to make purchases. The homepage includes a list of books available in the website by multiple sellers. Account holders can edit(eg: date published, authors names, etc) and list books for sale.

Technologies used: Ruby 2.6.4, Rails 5.4.7, RubyMine 2019, SQLite 3.29, JavaScript, HTML and CSS.
